Experiencia multidispositivo

Una experiencia adaptada al contexto de cada dispositivo

La navegación, jerarquía de contenido e interacción fueron ajustadas según el contexto de uso para mantener claridad y consistencia entre desktop, tablet y mobile.

Desktop

Mayor densidad de información y exploración simultánea para facilitar comparación y descubrimiento de productos.

Tablet

Balance entre visibilidad y simplicidad, optimizando navegación táctil y lectura de contenido.

Mobile

Priorización de acciones clave, filtros simplificados y navegación enfocada en rapidez y claridad.

Estrategia UX

Diseñando para distintos contextos de uso

La experiencia fue adaptada para responder a diferentes patrones de navegación, niveles de atención y formas de interacción según el dispositivo utilizado.

En desktop se priorizó exploración y comparación. En tablet se buscó equilibrio entre densidad y claridad. En mobile se simplificó la navegación para acelerar tareas clave.

Desktop

Mayor exploración simultánea, navegación expandida y comparación visual de productos.

Tablet

Optimización táctil y reorganización de módulos para mejorar lectura y navegación.

Mobile

Priorización de acciones rápidas, filtros simplificados y reducción de fricción.

Wireframes

Estructurando una experiencia flexible y escalable

Los wireframes permitieron definir jerarquías, comportamiento responsive y distribución modular del contenido antes de avanzar a interfaz visual.

Se trabajó una estructura adaptable para mantener consistencia entre dispositivos sin perder claridad ni eficiencia de navegación.

Decisión UX

Navegación simplificada para priorizar tareas clave

Se utilizó un menú colapsable incluso en desktop para reducir ruido visual y mantener el enfoque en búsqueda, categorías y descubrimiento de productos.

La navegación secundaria no representaba la acción principal del usuario, por lo que se priorizó claridad de interfaz y acceso rápido al contenido relevante.

Sistema visual

Un sistema visual consistente entre dispositivos

Se desarrolló un sistema visual adaptable para mantener coherencia, legibilidad y consistencia entre desktop, tablet y mobile.

Color

Se definió una paleta enfocada en contraste, claridad visual y consistencia de marca.

Tipografía

Jerarquías tipográficas adaptables para mejorar lectura y escaneo de contenido.

Componentes

Componentes reutilizables para mantener consistencia visual y escalabilidad.

Hallazgos de usabilidad

Identificando fricción y oportunidades de mejora

Las pruebas de usabilidad permitieron detectar problemas relacionados con jerarquía visual, navegación y claridad en acciones clave dentro de la experiencia responsive.

  • Confusión en navegación secundaria
  • Exceso de elementos visibles en desktop
  • Problemas de jerarquía en mobile
  • Dificultad para identificar acciones prioritarias

Solución final

Una experiencia más clara, flexible y enfocada en la decisión

La solución final prioriza navegación intuitiva, consistencia responsive y acceso rápido a información relevante para facilitar exploración y comparación de productos.
